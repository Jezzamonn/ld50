A game for Ludum Dare 50 "Delay the inevitable" by Jezzamon

What's going on:

A kitten has noticed our peaceful mouse village! We must try to distract it as long as possible!

Controls:

  • WASD / Arrow Keys to move
  • Space to Dash
  • Space near an object to pick it up
  • Space while holding an object to throw it

Bring stuff to the kitten to delay its inevitable destruction!

Throwing certain objects at other things might have useful results!
